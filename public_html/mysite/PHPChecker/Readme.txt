Hi, if you want to check the syntax of a php file you can use this 

To check a single php file
- Run the PHPCheckerRunner.bat
- Enter the file path
- The result will be in the log.txt file

To check all files in 1 folder and all sub folders
- Run the PHPCheckerRunner.bat
- Enter the path of the folder
- The result will be in the log.txt file
